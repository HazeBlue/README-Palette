## 🎯 设计原则

### 1. 对比度
所有主题的文字颜色与背景色对比度均符合 WCAG AA 标准（至少 4.5:1）。

### 2. 阴影系统
- **标准阴影**：`rgba(0, 0, 0, 0.05)` 或主题色透明度 0.1
- **悬停阴影**：`rgba(0, 0, 0, 0.1)` 或主题色透明度 0.2

### 3. 圆角系统
- **卡片圆角**：`20px`
- **按钮圆角**：`12px` / `25px`（圆形按钮）
- **输入框圆角**：`12px`

### 4. 渐变应用
- 主要用于按钮背景和 Logo 文字
- 使用 135 度角渐变，营造现代感

### 5. 过渡动画
所有交互元素使用 `transition: all 0.3s ease` 实现平滑过渡。

## 🔧 使用方法

在 CSS 中通过 `data-theme` 属性切换主题：

```css
body[data-theme="mint"] {
    --primary: #10b981;
    /* ... 其他变量 ... */
}
```

或在 JavaScript 中动态切换：

```javascript
document.body.setAttribute('data-theme', 'mint');
```

---

## 📝 色彩命名规范

所有颜色变量使用语义化命名：

- `--primary`：主色调
- `--secondary`：次要色
- `--bg-primary`：主背景色
- `--bg-secondary`：次背景色
- `--bg-card`：卡片背景色
- `--text-primary`：主文字色
- `--text-secondary`：次文字色
- `--border`：边框色
- `--shadow`：阴影色
- `--accent`：强调色
- `--gradient-1/2/3`：渐变方案
